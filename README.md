# Learning Workflow

This repository is my structured learning workspace for growing into a stronger **Data Analytics Engineer** with a solid foundation in analytics, data modeling, and scalable data operations.

I use this space to organize what I learn, what I build, and how each topic connects back to my real-world work experience and future career goals.

My current focus is on strengthening practical capabilities in:

- SQL for analytics engineering
- Databricks and PySpark
- Data warehouse/data mart modeling
- Data quality and documentation
- Reusable transformation logic
- Resume-ready learning summaries

This is not just a study log.

The goal of this repository is to turn learning into **career assets** by documenting:

- what I learned

## Folder Guide

summaries/
Structured markdown summaries for each course, tutorial, or project.

Each summary is written using a standard template and includes:
- What I learned
- What I built
- Why it matters
- How it connects to my past work
- Resume bullet draft

notebooks/
Exported notebooks from Databricks or notebook-based experiments.

sql/
SQL practice files, transformations, validation queries, and analytics logic.

python/
Python and PySpark practice scripts related to data processing and workflow automation.

templates/
Reusable templates for summaries and project documentation.
